# 📚 Student Result Management System (C++)

## 📌 Description

This is a **menu-driven C++ program** that manages student records.
It allows the user to add students, enter marks, calculate results, search students, and find the class topper.

The program is built using **Object-Oriented Programming (OOP)** concepts and **STL vector container**.

---

## 🚀 Features

* Add new student
* Enter marks of 3 subjects
* Calculate total and percentage
* Display all students
* Search student by roll number
* Find topper of the class

---

## 🛠 Technologies Used

* C++
* Object Oriented Programming (OOP)
* STL Vector
* Algorithm Library (`max_element`)

---

## 📋 Menu Options

1. Add Student
2. Add Marks of 3 Subjects
3. Calculate Result
4. Display All Students
5. Search Student
6. Show Topper
7. Exit

---

## ▶ How to Run

1. Compile the program:

g++ student_result.cpp

2. Run the program:

./a.out
